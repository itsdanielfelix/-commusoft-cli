# Commusoft CLI

[![Repo Size][ico-github-repo-size]][link-github-repo-size]

# Install via [Homebrew][link-homebrew]

```shell
# add custom tap
brew tap itsdanielfelix/tap

# install formulae
brew install commusoft
```

## Update

```shell
# run upgrade command
brew upgrade commusoft
```
